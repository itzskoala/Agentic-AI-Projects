# 🧠 Agentic File Manager

A simple AI-powered file manager that uses OpenAI to summarize and analyze files.

---

## 🚀 Setup

To run this project, you’ll need your own OpenAI API key.

### 1. Get an API Key
Sign up or log in on the OpenAI platform and generate an API key.

### 2. Create a `.env` File
In the root directory of the project, create a file named:

.env

### 3. Add Your API Key
Inside the `.env` file, add:

OPENAI_API_KEY=your_api_key_here

> ⚠️ **Important:** Never share your API key or commit your `.env` file to version control.

---

## ▶️ Running the Program

Once your API key is set up, run:

python3 main.py

---

## 📦 Requirements

Install dependencies (if applicable):

pip install -r requirements.txt

---

## 🧠 Notes

- The application automatically loads your API key from the `.env` file.
- Make sure your `.env` file is in the root directory.
- If something isn’t working, double-check your API key and environment setup.

---

## 📁 Project Idea

This project demonstrates how to:
- Load and process files programmatically
- Use AI to summarize and extract insights
- Build a simple agentic workflow using APIs
